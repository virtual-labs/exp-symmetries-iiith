Instructions
  • This experiment is designed to teach various symmetries that exist in crystal lattices.
  • The left side of the UI shows the current Lattice being rendered. One may use the dropdown to switch to other lattices.
  • The 'Select Atoms' button enables the select atom mode in which the user can select the atoms of interest which will later be used to verify planar symmetry(if three points chosen), point symmetry (if one point chosen) and rotational symmetry if two poitns chosen.
  • After selecting the points of interest, the user may click 'Point symmetry' to visualize the point symmetry across the one point selected. One may use the slider to rotate the system of atoms across the axis of the two points selected. One may use the 'Plane Symmetry' button to visualise the planar symmetry across the plane formed by the three selected points.
  • Once the points are selected, 'Check Symmetry' button can be used to check if such a symmetry exists in the lattice provided. The function will verify for the symmerty depending on the points chosen. Like if one point is selected then it will verify point symmetry, two points for rotational symmetry and three points for planar symmetry.
