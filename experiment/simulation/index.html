Instructions
  • The arrows to the sides of the name of the lattice should be used for navigating between different lattices
  • Left click and drag to move the camera around. You can use the mouse scroll wheel to zoom in or zoom out.
  • To select atoms, click on the select atoms button. It is to be noted that the select atoms button is a toggle switch. Once you have made your choice, click on the check vectors button to get the result of your choice.
  • You may use the select region button to visualize the convex polyhedron constructed by the selected atoms. Note that this function will only work if the selected atoms form a valid convex polyhedron
  • Please select the set of atoms(in any order), all of which are a part of a unique primitive or non primitive lattice.
